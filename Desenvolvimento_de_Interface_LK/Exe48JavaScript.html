Exemplo 48 - Uso do operador Ternário com JavaScript

O que é o Operador Ternário?

O operador ternário é um atalho para a estrutura if / else tradicional. Ele permite avaliar uma condição e retornar um valor em apenas uma linha de código.

condição ? valor_se_verdadeiro : valor_se_falso;
