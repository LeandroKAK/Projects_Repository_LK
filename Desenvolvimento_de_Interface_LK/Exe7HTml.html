Exemplo 7 - HTML Listas Ordenadas e Não Ordenadas

Este exemplo mostra como criar listas ordenadas e não ordenadas incluindo subitens(listas aninhadas)

Descrição das tags utilizadas na criação das listas.

  • <ol>:Define uma lista ordenadas (com números, letras e/ou algarismos romanos)
  • <ul>:Define uma lista não ordenada (com marcadores, com pontos e quadrados)
  • <li>:Define um elemento na lista

Exemplo de lista ordenada <ol>

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
    4. Item 2.4
    5. Item 2.5
  3. Item 3
    1. Item em romanos
    2. Item em romanos
    3. Item em romanos
    4. Item em romanos
      1. Item em romanos minusculo
      2. Item em romanos minusculo
      3. Item em romanos minusculo
      4. Item em romanos minusculo
  4. Item 4
    1. Item com letras
    2. Item com letras
    3. Item com letras
    4. Item com letras
    5. Item com letras
    6. Item com letras
      1. Item com letras minusculas
      2. Item com letras minusculas
      3. Item com letras minusculas
      4. Item com letras minusculas
    7. Item com letras
  5. Item 5

Exemplo de lista não ordenada <ul>

  • Item 1
  • Item 2
    • subitem 2
    • subitem 3
    • subitem 4
      • Subitem 1
      • Subitem 2
      • Subitem 3
      • Subitem 4
      • Subitem 5
    • subitem 5
  • Item 3
  • Item 4
