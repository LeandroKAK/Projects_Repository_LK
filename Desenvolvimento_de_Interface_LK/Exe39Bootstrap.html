Formularios com bootstrap

Em termos gerais, formularios web sao ferramentas que permitem a interação direta do usuario com o sistema. Eles funcionam como uma ponte de comunicação coletando dados fornecidos pelos usuarios da aplicação.

O bootstrap simplifica drasticamente a criação de formularios, atuando diretamente em quatro pilares fundamentais: bordas, tipografia, efeito visual e organização. Abaixo exemplificamos um formulario criado em bootstrap.
