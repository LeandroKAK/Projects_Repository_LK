Exemplo 33 HTML CSS - FlexBox

O FlexBox é o módulo de layout CSS que foi projetado para facilitar a organização de elementos html de forma responsiva e flexivel. Ele permite alinhar e distribuir espaços de forma automática dentro de um container. Ele é muito utilizado na criação e construção de layouts complexos"

Exemplo das principais propiedades:
