O que é o Grid do Bootstrap?

O Grid System (Sistema de Grade) é o coração do Bootstrap. Trata-se de uma estrutura matemática baseada em Flexbox utilizada para organizar, alinhar e posicionar o conteúdo de uma página web de forma totalmente responsiva.

A lógica do Grid funciona dividindo a tela horizontalmente em um layout invisível de 12 colunas imaginárias. A partir dessa divisão, o desenvolvedor pode decidir exatamente quantas colunas cada elemento do site deve ocupar em diferentes tamanhos de tela (como celulares, tablets e computadores).

A Regra de Ouro da Estrutura (Os 3 Pilares):

Para o Grid funcionar corretamente, o HTML precisa seguir rigorosamente uma hierarquia de três níveis:

  1. Container (.container ou .container-fluid): É a camada mais externa. Ele centraliza a página e define as margens de respiro nas laterais.
  2. Linha (.row): Fica obrigatoriamente dentro do container. A linha funciona como um "embrulho" (wrapper) que agrupa as colunas e impede que elas transbordem para fora do layout.
  3. Coluna (.col-*): Fica obrigatoriamente dentro da linha. É aqui dentro que o conteúdo real do site (textos, imagens, formulários) deve ser inserido.

Como funciona a matemática dos 12 espaços?

Como a linha sempre soma 12 espaços, você define o tamanho combinando a classe .col- com o número desejado:

  • Duas colunas iguais lado a lado? Use duas .col-6 (6 + 6 = 12).
  • Três colunas iguais? Use três .col-4 (4 + 4 + 4 = 12).
  • Uma barra lateral menor e um conteúdo principal maior? Use uma .col-3 e uma .col-9 (3 + 9 = 12).
Dica de Responsividade: Você pode combinar tamanhos adicionando os breakpoints do Bootstrap. Por exemplo, a classe class="col-12 col-md-6" diz ao navegador: "Em celulares, oculpe a tela inteira (12), mas a partir de telas médias/computadores (md), ocupe apenas metade (6)".
