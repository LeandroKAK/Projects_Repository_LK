Exemplo 4 HTML - Estrutura Geral

Este exemplo mostra a estrutura minima de um documento HTML, A seguir são descritas as principais tags utilizadas nessa estrutura geral

  • <DOCTYPE html>:É uma instrução HTML, essa tag define o tipo do documento html.
  • <html>: É o elemento central do documento, é o container do documento html, ele engloba toda a página
  • <head: Essa tag contémos metadados que serão interpretados pelo navegador, tai como titulos e metatags de codificação.>
  • <body: Esse bloco é responsável por deixar visível o conteúdo destinado aos usuários. Ou seja todo o contéudo do documento html>
