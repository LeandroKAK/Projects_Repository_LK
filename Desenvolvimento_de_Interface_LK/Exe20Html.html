CSS

O CSS é o acrônimo de folhas de folhas de estilo em cascata, ele é usado para estilizar documentos HTML. Ele deve ser aplicado às tags HTML, usado para formatações diversas do documento, tais como: mudança de cores, fontes e posicionamentos dos elementos html.

Tipos seletores - seletor de Tipo
